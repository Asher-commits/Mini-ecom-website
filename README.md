# Mini-ecom-website

A simple e-commerce frontend that allows users to sign up, log in, search products, add to favorites, and manage a shopping cart using JavaScript, LocalStorage, and the Fake Store API

Features

1.User Authentication

- Sign up and login functionality.

- Display logged-in user's first name.

2.Product Display

- Shows trending products (4 random items from API).

- Trending products update every 10 seconds.

- Search products by name with live results.

3.Favorites

- Add/remove products to favorites.

- Persist favorites using LocalStorage.

4.Shopping Cart

- Add/remove items from cart.

- Checkout functionality calculates total price.

- Persist cart using LocalStorage.

5.Responsive UI

- Sections for home, search results, favorites, and cart.

- Show/hide sections dynamically based on user actions.
